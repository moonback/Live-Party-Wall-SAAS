# Sons pour les Battles

Ce dossier contient les fichiers audio utilisés pour les résultats de battles.

## Fichiers requis

### `victory.mp3`
Son joué lorsqu'une photo gagne une battle.
- Format recommandé : MP3
- Durée recommandée : 2-4 secondes
- Style : Son de victoire, célébration, applaudissements, ou fanfare

### `defeat-tie.mp3`
Son joué en cas d'égalité ou de défaite.
- Format recommandé : MP3
- Durée recommandée : 1-3 secondes
- Style : Son neutre ou discret pour les égalités

## Sources recommandées

Vous pouvez trouver des sons gratuits sur :
- [Freesound.org](https://freesound.org/) - Sons libres de droits
- [Zapsplat](https://www.zapsplat.com/) - Bibliothèque de sons gratuits
- [Mixkit](https://mixkit.co/free-sound-effects/) - Effets sonores gratuits

## Notes

- Les fichiers doivent être au format MP3 pour une meilleure compatibilité
- Les sons sont joués automatiquement lors des résultats de battles
- Le volume peut être ajusté dans `utils/soundService.ts`
